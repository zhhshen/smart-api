### Smart Api
---

#### 描述

```
根据项目中的mock数据，生成apiConfig文件，并自动替换指定文件中的api, 同时可以撤销相应的操作
```

#### 安装

```bash
npm install sapis-cli -g
```

#### 配置项

```
命令：
  sapis init [name]  init a config file
  sapis replace      replace api
  sapis reset        reset api

选项：
  --version, -V  显示版本号                                               [布尔]
  --help, -h     显示帮助信息                                             [布尔]
```
